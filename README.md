# 📌 Overview

Huellita PetShop is a responsive front-end e-commerce prototype developed as a portfolio project. It focuses on delivering a modern shopping experience while emphasizing responsive design, accessibility, SEO, performance, and maintainable code.

The project follows a mobile-first approach and aims to provide a lightweight and user-friendly interface for browsing pet products.

---

# ✨ Features

| Module          | Description                              |
| --------------- | ---------------------------------------- |
| 🏠 Home         | Hero section, categories, promotions     |
| 🛍 Catalog      | Responsive product grid                  |
| 🔎 Search       | Filters by species, category, and price  |
| 📦 Details      | Product information and related products |
| 🛒 Cart         | Dynamic cart with quantity management    |
| 💳 Checkout     | Responsive validated form                |
| 📄 Confirmation | Purchase summary                         |
